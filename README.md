# 🎮 Tic-Tac-Toe Game 🕹️
**What It Does:**
This is a stylish, interactive Tic-Tac-Toe game with win detection, colorful announcements, and a confetti effect.

## 🛠️ How to Play:
1. **Open the Game**: Launch `index.html` in a web browser.
2. **Make a Move**: Click on an empty cell to place your mark (X or O).
3. **Take Turns**: Players alternate placing X and O.
4. **Win the Game**: Get 3 in a row (horizontal, vertical, or diagonal) to win.
5. **Draw**: If all cells are filled with no winner, it's a draw.

## 📁 Files
- `index.html`: Game board structure
- `styles.css`: Game styling
- `script.js`: Game logic and interactivity
- `confetti.js`: Confetti effect library
- `font.css`: Custom font declaration
- `font.otf`: Custom font file (not included)

Enjoy your Tic-Tac-Toe! 🏆
